Privacy Policy for TG Link Finder

This Privacy Policy explains how TG Link Finder handles information when users use the extension.

Effective date: April 13, 2026

Contact email: congrong512123@gmail.com

1. Purpose of the Extension

TG Link Finder is a Chrome extension that helps users discover Telegram group, channel, and invite links related to specific keywords or target websites by searching Google result pages and extracting relevant Telegram links for review, copy, and export.

2. Information the Extension Processes

The extension may process the following information in order to provide its core functionality:

  • User-entered search keywords.
  • User-selected search settings, such as target website, page count, and Google domain.
  • Telegram links found on Google search result pages.
  • Local extension state and preferences needed to restore the user interface and search progress.
The extension is not designed to collect phone numbers, contacts, private messages, or Telegram account credentials.

3. How Information Is Used

Information processed by the extension is used only to provide the extension’s core features, including:

  • Running Google-based searches requested by the user.
  • Extracting Telegram links from search result pages.
  • Displaying search progress and results in the Chrome side panel.
  • Allowing users to copy or export extracted links.
  • Saving local preferences and temporary session data for convenience.

4. Storage and Retention

TG Link Finder stores search settings, search state, and extracted results locally in the user’s browser through Chrome extension storage. This information is kept only to support the extension’s functionality and remains under the user’s control inside the browser environment.

5. Data Sharing

TG Link Finder does not sell personal data.

TG Link Finder does not share user data with third parties for advertising or marketing purposes.

The extension may access Google search result pages and, if optional account or membership features are enabled in the future, may communicate with related service endpoints only as necessary to provide those features.

6. Remote Code

The extension does not use remotely hosted executable code. Its executable logic is packaged within the extension bundle.

7. User Control

Users can control their use of the extension by:

  • Choosing what keywords and websites to search.
  • Clearing search state and results from the extension interface.
  • Removing the extension from Chrome at any time.

8. Security

Reasonable efforts are made to limit data handling to what is necessary for the extension’s stated purpose. However, no method of electronic storage or transmission is completely secure, and absolute security cannot be guaranteed.

9. Changes to This Policy

This Privacy Policy may be updated from time to time to reflect product, legal, or operational changes. The updated version will be posted on this page with a revised effective date.
